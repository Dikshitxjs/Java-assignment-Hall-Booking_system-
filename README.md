# Hall Booking Management System

This is a Java-based application for managing hall bookings, payments, and maintenance.

## Project Structure

- **src/main/**: Main application entry point
- **src/model/**: Data models (User, Customer, Scheduler, etc.)
- **src/gui/**: GUI components and frames
- **src/service/**: Business logic services
- **src/util/**: Utility classes
- **src/constants/**: Application constants
- **data/**: Data storage files
- **report/**: Documentation and reports
